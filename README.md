# discordchatbot

## 주 기능
1. 10분마다 자동으로 랜덤 명언 출력
2. 명령어를 사용해 명언 출력
3. 명언 데이터베이스에 나만의 명언 추가

## 랜덤 명언 출력 봇 사용법

안녕하세요! 저는 랜덤 명언 출력 봇입니다. 사용법은 다음과 같습니다:

1. **랜덤 명언 받기**
   - `/명언` 명령어를 사용하여 랜덤 명언을 받습니다.

2. **명언 추가**
   - `/명언추가 author quote` 명령어를 사용하여 명언을 추가할 수 있습니다.
   
3. **사용법 확인**
   - 채팅에 아무거나 입력하여 사용법을 다시 확인할 수 있습니다.
